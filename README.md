# zoe的小程序公共组件

## 项目运行

```
git clone https://github.com/chenzeze/common_component.git

(使用微信开发者工具运行)

```

## 项目结构

```

1、公共组件 com 

2、云函数 clouds

3、公共方法 utils 

```
